## 📄 About ai-resume-screening-system

This application helps you speed up the hiring process. It uses artificial intelligence to read resumes stored in AWS S3. Then, it sorts candidates based on the rules your HR team sets. It saves the results in a simple SQLite database. You will also get automatic notifications by email and WhatsApp. This system is designed to work like a real recruitment tool with secure cloud and database features.

You do not need any programming skills to run it on a Windows computer.

## 🗃️ Step 5: Understanding How It Works

Once running, the app performs these tasks automatically:

1. Downloads resumes from your AWS S3 bucket.  
2. Uses AI models to evaluate each resume according to the rules set by your HR department.  
3. Classifies candidates as "Suitable," "Consider," or "Reject."  
4. Stores the screening results in an easy-to-access SQLite database on your computer.  
5. Sends automatic notifications via email and WhatsApp to notify recruiters and candidates.  

You can review results anytime by opening the app or accessing the database file directly.
